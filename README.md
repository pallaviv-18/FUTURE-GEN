# FUTURE-GEN
🌐 FutureGen – Futuristic AI & Technology Website

FutureGen is a modern, multi-page futuristic website built using HTML, CSS, and JavaScript. It showcases emerging technologies like Artificial Intelligence, Robotics, Smart Cities, and Green Energy with a clean, responsive, and futuristic UI design.

The project is designed for learning and demonstration purposes, featuring a professional layout with glassmorphism effects, animations, and interactive elements.

✨ Features
🤖 AI, Robotics & Smart City themed pages
🌌 Futuristic UI with glassmorphism design
📱 Fully responsive for mobile, tablet & desktop
📊 Animated counters and progress bars
🧠 AI chatbot simulation (JavaScript-based)
💡 "Did You Know?" fact generator
⚡ Multi-page navigation system
🔍 SEO optimized structure
🌍 Clean and modern UI design
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
